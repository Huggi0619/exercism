# Exercism Solutions (Python)

This repository contains my solutions to various programming exercises on [Exercism](https://exercism.org). 

## Purpose
The goal of these exercises is to improve my proficiency in Python by solving problems of increasing complexity, ranging from basic syntax to advanced concepts like decorators, generators, and object-oriented programming.

## Structure
The repository is organized by exercise name. Each folder typically contains:
* `solution.py`: My implementation of the problem.
* `README.md`: The exercise description and requirements (provided by Exercism).

## Key Learning Areas
* **Control Flow:** Loops, conditionals, and exception handling.
* **Data Structures:** Efficient use of lists, dictionaries, sets, and tuples.
* **Functional Programming:** Lambda functions, map/filter, and list comprehensions.
* **Testing:** Using `pytest` to ensure code correctness against predefined test suites.

## About Exercism
Exercism is a free, open-source coding platform that offers interactive programming tracks. It focuses on deep learning through practice and mentor feedback.
